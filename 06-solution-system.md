# 拡張性を考慮したシステム設計

## 複数ハウスメーカーへの展開を見据えて

### システムの特徴
1. **UI/UXの汎用化**
   - 業界共通の業務フローに基づく標準化

2. **マルチテナント構成**
   - データとロジックの分離による企業ごとのカスタマイズ容易性

3. **デザイン要素の分離**
   - 企業ロゴや色などのブランド要素を設定ファイルで管理

4. **API設計**
   - 将来的な他システムとの連携を見据えたインターフェース
